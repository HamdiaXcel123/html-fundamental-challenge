SOLID is actually an acronym,with each letter representing one of the first five core principles
of object-oriented design (OOD).These were brought together and popularized by Robert C. Martin - better known as
“Uncle Bob” in the programming community, a figure whose work has shaped the best practices followed by countless developers around the world.
Each of these principles addresses a common pain point in software development, offering guidance on how to avoid pitfalls such as tangled, rigid code that’s hard to modify or extend.
